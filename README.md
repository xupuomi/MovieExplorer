# Movie Explorer

Explore the movie catalog to find your next watch! Filter by genre & media type and sort by popularity, alphabetical order and more!

## How to Run

### (Option 1) Run Locally
1. `npm i`
2. `npm run start`
3. Open http://localhost:3000/MovieExplorer and enjoy!

### (Option 2) Use Deployed Site
https://xupuomi.github.io/MovieExplorer/

### Tools Used:
  - React Router
  - Axios
  - TypeScript
